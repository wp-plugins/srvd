How to use Srvd Wordpress plugin

1. Connect your account by entering your API code from Srvd into the box above.

2. Create a placement in your Srvd account.

3. Navigate to any page or post within your Wordpress account.

4. In the WYSIWYG editor on the “Visual” tab you will see a Srvd logo. Click this and you will be able to choose a placement. This will be positioned in the exact location your cursor is placed.

5. That’s it. You have just used Srvd Wordpress plugin to add a new placement to your Wordpress post or page.

If you have any issues please click the help button on any page within the Srvd app.
